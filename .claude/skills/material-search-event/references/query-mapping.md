# 自然语言到标签映射

## 常见映射

- “反杀” → `power_dynamic: 翻转`
- “告别” → `event_type: 告别`
- “催泪” → `reader_effect: 催泪`
- “不煽情” → 倾向 `technique: 留白` / 克制表达
- “对话难写” → `dialogue_type` + `relationship`

## 解析原则

1. 先抓主事件动作
2. 再补关系 / 情绪 / 技法
3. 映射不唯一时，优先保留更强语义信号

## 放宽顺序

无结果时，优先去掉：

1. 最弱的技法维度
2. 再去掉风格或附加情绪维度
3. 最后才放宽核心事件动作
