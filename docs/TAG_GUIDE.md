# 标签使用指南

> 供 LLM 在事件标注（`novel-scenes`）时参考，目标：**提升标注一致性和区分度**。
> 本文件由人工维护，不由 skill 自动生成。

## 总原则

1. **先读后判** — 必须阅读事件原文再打标签，不可凭章节标题猜测
2. **多选求精** — 列表字段（`[]`）允许多值，但每个值都必须有原文支撑
3. **区分主次** — 多值时把最突出的放第一位
4. **宁缺勿滥** — 无信号时写空列表 `[]`，不要硬凑

## A. 事件内容层

### event_type — 事件类型

**判断依据**：事件的核心行动是什么？

| 易混淆对 | 区分标准 |
|----------|----------|
| 对决 vs 争吵 | 对决有胜负/生死结果；争吵是言语冲突，未必分胜负 |
| 密谋 vs 谈判 | 密谋是背着某方的暗中策划；谈判是双方公开博弈 |
| 考验 vs 修炼 | 考验有明确通过/失败标准（闯关、测试）；修炼是自我提升过程 |
| 仪式 vs 祭祀 | 仪式更广义（结盟、加冕）；祭祀特指祭奠/献祭 |
| 潜入 vs 逃亡 | 潜入是主动进入敌方领域；逃亡是被动脱离危险 |
| 赌局 vs 谈判 | 赌局有随机/博弈机制（拍卖、下注）；谈判靠说服 |
| 回忆 vs 梦境 | 回忆是角色主观追忆真实过去；梦境是非真实的意象化事件 |
| 日常 vs 独处 | 日常有多人互动但无核心冲突；独处是单人事件 |

### conflict — 冲突类型

**判断依据**：冲突的根源是什么？

| 标签 | 识别特征 | 典型事件 |
|------|----------|----------|
| 价值观 | 双方都觉得自己是对的 | 正道vs魔道的理念之争 |
| 误会 | 信息不对称导致，真相揭开即可消解 | 以为对方叛变 |
| 利益 | 明确的资源/好处争夺 | 争夺灵药、地盘 |
| 道德 | "该不该做"的内心挣扎 | 为救一人牺牲多人 |
| 人与自我 | 没有外部对手，矛盾在自己内心 | 自我怀疑、欲望与理性拉扯 |
| 人与命运 | 抗争的是不可抗力/天命 | 预言/天劫/宿命 |
| 人与社会 | 个体vs制度/阶层/群体压力 | 反抗门规、挑战阶级 |
| 人与自然 | 生存级的环境威胁 | 荒野求生、天灾逃生 |
| 资源争夺 | conflict=利益 的子类，更具体指有限资源 | 灵石矿/秘籍/唯一名额 |
| 新旧冲突 | 改革者vs守旧者 | 废除旧制、宗门改革 |

**与 `人与自我` 的边界**：如果角色面对外部对手但同时在内心挣扎，外部冲突用具体类型（利益/生死等），`人与自我` 只在**没有外部对手、纯内心矛盾**时使用。

### stakes — 赌注

**判断依据**：如果这个事件的结果最坏，角色会失去什么？

- `道心`：修炼/信仰根基动摇，适用于修真/玄幻类，角色可能走火入魔或修为倒退
- `底线`：人格/原则被突破，"做了这件事就不是我了"
- `族群存续`：关乎家族/种族/门派生死存亡，比"世界存亡"更聚焦

## B. 人物层

### archetype — 角色原型

来自荣格原型理论 + 坎贝尔英雄之旅 + 沃格勒七原型。**标注角色在这个事件中表现出的原型**，同一角色不同事件可能展现不同原型面。

| 原型 | 核心驱动 | 识别信号 |
|------|----------|----------|
| 英雄 | 证明自己的价值 | 面对挑战、正面迎敌 |
| 导师 | 传授/引导 | 给主角建议、训练、指路 |
| 门槛守卫 | 测试是否配得上 | 拦路、设门槛、考验 |
| 信使 | 打破日常 | 带来消息/变化/任务 |
| 变形者 | 立场不明 | 忽敌忽友、让人摸不透 |
| 影子 | 主角的黑暗面 | 与主角价值观对立但有相似之处 |
| 骗术师 | 打破规则 | 搞笑、出人意料、不走寻常路 |
| 天真者 | 追求安全与幸福 | 对世界抱有单纯信任 |
| 反叛者 | 颠覆秩序 | 挑战权威、破坏规则 |
| 统治者 | 掌控 | 制定规则、维护秩序、发号施令 |
| 照顾者 | 保护他人 | 自我牺牲、照顾弱者 |
| 殉道者 | 信念高于生命 | 以自身代价成就信仰/他人 |

### character_moment — 人物弧光时刻

**与 psychology 字段的联动**：

| character_moment | 对应的 psychology 字段 | 事件信号 |
|-----------------|----------------------|----------|
| 执念暴露 | obsession | 角色的长期追求公开化或失控 |
| 软肋被击 | soft_spot | 角色最在意的人/事被威胁或伤害 |
| 旧创复发 | tragedy_trigger | 类似过去创伤的情境再次出现 |
| 面具脱落 | contrast_habit | 伪装/人设被识破 |
| 对照时刻 | — | 两个角色形成镜像对比（同一选择不同答案） |

### power_dynamic — 权力位差

- `翻转`：**事件内**发生了位差逆转（原来强势的一方变弱势）
- `此消彼长`：不是单次翻转，而是持续的拉锯/博弈

## C. 情感层

### emotion — 情绪基调

**区分相似情绪**：

| 易混淆对 | 区分 |
|----------|------|
| 悲伤 vs 怀念 | 悲伤指向当下的痛；怀念指向过去的温暖中带着失落 |
| 绝望 vs 麻木 | 绝望还有痛感；麻木是情感已耗尽 |
| 紧张 vs 不安 | 紧张烈度高（即将发生冲突）；不安是低烈度的焦虑/预感 |
| 愤怒 vs 屈辱 | 愤怒是外向的攻击性；屈辱是内向的自尊受伤 |
| 挣扎 vs 无奈 | 挣扎还在反抗；无奈已经知道反抗无用 |
| 癫狂 vs 愤怒 | 癫狂已失去理性控制；愤怒仍有方向性 |

### tension — 张力值 (1-5)

| 值 | 含义 | 参考 |
|----|------|------|
| 1 | 平静日常 | 闲聊、散步、用餐 |
| 2 | 有轻微张力 | 暗流涌动、小摩擦、隐隐不安 |
| 3 | 中等冲突 | 正面争吵、中等规模战斗、重要决定 |
| 4 | 强烈冲突 | 生死关头、重大背叛、关键战役 |
| 5 | 极限张力 | 全书高潮、不可逆转折、生离死别 |

**校准**：全书 tension=5 的事件不应超过总事件数的 5%。如果标出太多 5，说明标准偏松。

### reader_effect — 期望读者感受

**与 emotion 的区别**：emotion 是角色的感受，reader_effect 是**读者**的感受。

- 角色绝望 → 读者可能是"揪心"或"窒息"
- 角色狂喜 → 读者可能是"爽感"或"会心一笑"
- 角色平静 → 读者可能是"怅然若失"或"治愈"

| 标签 | 触发机制 |
|------|----------|
| 拍案叫绝 | 精巧的反转/布局让人赞叹作者构思 |
| 欲罢不能 | 悬念钩子让人想翻下一页 |
| 如释重负 | 长期悬念/危机终于解除 |
| 唏嘘 | 物是人非、命运弄人、错过不可挽回 |

## D. 结构层

### plot_stage — 剧情阶段

基于三幕结构 + 节拍表（Save the Cat）。**按事件在全书中的位置判断**：

| 阶段 | 大致位置 | 核心功能 |
|------|---------|----------|
| 开篇 | 0-2% | 建立日常世界 |
| 第一幕-诱因 | 2-10% | 打破日常的事件 |
| 第一幕-发展 | 10-25% | 进入新世界/新状态 |
| 第二幕-对抗 | 25-50% | 升级挑战、积累力量 |
| 中点 | 50% | 虚假胜利/虚假失败 |
| 第二幕-低谷 | 50-75% | 一切崩塌/最大危机 |
| 第三幕-高潮 | 75-90% | 最终对决 |
| 第三幕-收束 | 90-98% | 收拾残局 |
| 尾声 | 98-100% | 新的日常 |

**网文特殊性**：超长网文（1000+章）可能有多个"幕"循环。此时按**大弧线**判断，或按当前"小高潮弧"内的相对位置判断。

### plot_function — 情节功能

| 易混淆对 | 区分 |
|----------|------|
| 伏笔埋设 vs 铺垫 | 伏笔是**隐性**的（读者未必注意到），铺垫是**显性**的（读者能感觉到在为什么做准备） |
| 转折 vs 反转 | 转折是剧情方向改变；反转是**认知颠覆**（之前信以为真的被推翻） |
| 并线 vs 呼应 | 并线是两条剧情线汇合到同一事件；呼应是遥远事件间的因果回响 |

## E. 技法层

### technique — 叙事技法

| 标签 | 识别特征 | 容易漏标的情况 |
|------|----------|----------------|
| 零度描写 | 克制的白描，不渲染情绪，以冷写热 | 作者写死亡/悲剧时用平淡语气 |
| 悬念钩子 | 章末/段末留未解之谜 | 事件戛然而止、突然切视角 |
| 草蛇灰线 | 看似闲笔实为钩子（反转铆合） | 不经意提到的物件/人物/细节 |
| 通感 | 跨感官描写 | "声音是温暖的""颜色有重量" |
| 陌生化 | 用全新视角描写熟悉事物 | 孩童视角看战争、外星人视角看人类 |
| 延宕 | 故意拖延关键信息的揭示 | 读者急切想知道但作者偏不说 |
| 预叙 | 预先暗示未来事件 | "多年以后…""他还不知道这是最后一次…" |
| 多声部 | 同一事件多角色/多立场交替发声 | 法庭戏、群戏、众说纷纭 |

**与 `info_delivery` 的关系**：technique 描述"怎么写的"（文学手法），info_delivery 描述"信息怎么传达给读者的"（功能性分类）。同一事件两者可并存且不冲突。

### dialogue_type — 对话类型

| 标签 | 关键特征 |
|------|----------|
| 指桑骂槐 | 表面聊A，实际攻击B/暗讽某人 |
| 激将 | 故意说反话/刺激对方以达到目的 |
| 打哑谜 | 双方心照不宣，第三方或读者可能不明白 |
| 遗言 | 临终/诀别事件的最后话语 |
| 言不由衷 | 说出来的和心里想的相反 |
| 沉默对峙 | 关键是"不说话"本身传递的信息 |

## F. 事件物理层

### setting — 空间类型

- `异空间`：不属于正常物理空间的场所（幻境、结界、精神世界、系统空间）
- `秘境`：存在于正常世界中但隐秘的场所（古墓、遗迹、禁地）
- `书院`：泛指教学场所（宗门、学校、学院、道场）

### scale — 事件人数规模

按**实际参与互动**的人数判断，不算背景路人：
- `独角戏`：1人
- `双人戏`：2人
- `三角`：3人（三角关系或三方博弈）
- `小团体`：4-10人
- `群戏`：10-50人
- `千军万马`：50+人或战争场面

## 标注质量检查清单

完成一批标注后，抽查 2 个事件确认：

- [ ] 同一批次内没有两个事件的标签组合完全相同
- [ ] `tension=5` 的事件不超过批次的 5%
- [ ] 没有连续 3 个以上事件的 `emotion` 完全一致
- [ ] `title` 是有语义的概括短语，不是"事件N"
- [ ] `summary` 概括了核心事件，不是首句截断
- [ ] 列表字段的空值 `[]` 有合理理由（独角戏无 relationship 等）
