### 问题
批量分析时 LLM 直接返回数组 `[...]` 而非对象 `{"chapters": [...]}`，导致 `'list' object has no attribute 'get'` 错误。

### 解决
- 增加 JSON 返回格式的兼容处理
- 支持数组直接返回和对象包裹两种格式
