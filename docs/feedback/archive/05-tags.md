## 标签字典警告

### 问题

```

```

### 解决
- 标签字典分级设计（一级分类 → 二级细分）
- 添加 `skip_tags` 参数跳过标签校验
- 新标签候选审核机制

- `src/novel_material/validation/validators.py`
- `data/tags/*.yaml`
