# 结构标签

结构标签描述小说的叙事结构、视角和时序处理。

---

## 一、叙事结构（15+）

| 标签值 | 说明 |
|--------|------|
| 三幕式 | 经典三幕结构（铺垫→对抗→解决） |
| 英雄之旅 | 坎贝尔英雄之旅12阶段 |
| 五幕式 | 莎士比亚五幕结构 |
| 环形结构 | 首尾呼应的环形叙事 |
| 多线叙事 | 多条叙事线索并行 |
| 单线叙事 | 单一叙事线索 |
| 章回体 | 章回体（传统说书结构） |
| 单元剧 | 每章/每几章为一个独立故事 |
| 编年体 | 按时间顺序编年叙事 |
| 传记体 | 人物传记式叙事 |
| 日记体 | 以日记形式叙事 |
| 书信体 | 以书信往来形式叙事 |
| 嵌套叙事 | 故事中的故事 |
| 碎片化 | 碎片化叙事 |
| 非线性 | 非线性叙事（时间线打乱） |

---

## 二、视角（10+）

| 标签值 | 说明 |
|--------|------|
| 第一人称 | "我"的视角 |
| 第二人称 | "你"的视角（极少使用） |
| 第三人称全知 | 全知全能的上帝视角 |
| 第三人称限制 | 跟随某一角色的有限视角 |
| 第三人称多视角 | 在多个角色视角间切换 |
| 双主角视角 | 两个主角交替叙事 |
| 多主角视角 | 三个及以上主角交替叙事 |
| 旁观者视角 | 以旁观者/旁白视角叙事 |
| 群像视角 | 群像叙事，无明确单一主角 |
| 伪纪录片 | 伪纪录片/访谈形式 |

---

## 三、时序处理（10+）

| 标签值 | 说明 |
|--------|------|
| 顺叙 | 按时间顺序叙事 |
| 倒叙 | 先讲结果再回述过程 |
| 插叙 | 在叙事中插入回忆/补充 |
| 补叙 | 事后补充说明 |
| 预叙 | 提前透露未来事件 |
| 时空交错 | 时间线交错叙事 |
| 平行叙事 | 多个时间线平行推进 |
| 时间跳跃 | 大幅度的时间跳跃 |
| 闪回 | 闪回到过去 |
| 时间循环 | 时间循环/轮回 |
