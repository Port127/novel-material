# 小说标签体系

> 本文档定义 novel-material 项目的完整标签体系，参考起点、番茄、飞卢、七猫等主流小说平台。
> 
> **设计原则**：宁多勿少，宁可分类过度也不要不够。未使用的标签不影响系统运行。
