# 进度与反馈优化

## 终端进度展示

### 问题
运行 pipeline 时看不到任何反馈，不知道进度、运行到了哪一步、健康状态和运行时间，无法监控和优化。

### 解决
- 实现了 7 阶段进度条展示
- 添加了章级分析的具体进度（如"第 31-40 章"）
- 每阶段完成后显示耗时统计

### 相关文件
- `src/novel_material/pipeline/progress.py`

### 归档日期
2026-05-10
