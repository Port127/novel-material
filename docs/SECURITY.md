# Novel Material 安全性

## 数据安全

### 素材存储

- 素材原文存储在本地仓库
- 无敏感数据（小说素材为公开或授权内容）
- Git 版本控制提供历史追溯

### 索引安全

- YAML 文件无执行能力
- 无用户个人信息存储
- 无外部 API 调用

## 访问控制

### 本地访问

- 仅通过 Claude Code skills 操作
- 无远程访问接口（M4 规划）

### 跨项目访问

- 小说项目通过相对路径读取
- 只读访问，不直接修改素材库

## Skill 安全

### 输入边界

- 文件路径验证（禁止访问外部目录）
- 参数类型检查
- 文件大小限制（大文件自动分段）

### 输出边界

- 仅写入 `data/` 目录
- 无外部网络调用
- 无进程执行

## 未来安全考虑

### API 层（M4）

若开放 API：
- 需要认证机制
- 需要速率限制
- 需要审计日志
