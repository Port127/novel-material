# 频道分类

频道是小说的最高层分类，决定内容的主要受众方向。

## 频道列表

| 标签值 | 说明 | 参考平台 |
|--------|------|---------|
| 男频 | 面向男性读者，以升级、战斗、权力、财富为核心 | 起点男频、番茄男频、飞卢 |
| 女频 | 面向女性读者，以情感、关系、成长为核心 | 起点女频、番茄女频、晋江 |
| 中性 | 男女通吃，以故事性、悬疑、世界观为核心 | 出版书、实体文学 |

## 各平台对比

| 平台 | 频道划分方式 |
|------|-------------|
| 起点 | 男频道道 / 女频道道（独立域名） |
| 番茄小说 | 男生 / 女生 / 有声书 / 出版书 |
| 七猫 | 男生 / 女生 / 图书 |
| 飞卢 | 男频同人 / 男频原创 |
| 晋江 | 女频为主（言情、耽美、无CP） |

## 选择依据

- **男频**：主角为男性，以事业/力量/权力/财富成长为主线
- **女频**：主角为女性，以情感关系/人物关系/个人成长为主线
- **中性**：无明确性别倾向，或以故事/悬疑/思辨为核心
