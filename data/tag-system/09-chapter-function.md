# 章节功能标签

章节功能标签描述章节在整部小说叙事中的功能定位。

---

## 一、开局功能（8+）

| 标签值 | 说明 | 典型章节 |
|--------|------|---------|
| 开局困境 | 主角陷入困境，建立紧迫感 | 第1章 |
| 悬念引入 | 引入核心悬念，吸引读者 | 第1-2章 |
| 人物亮相 | 主角/重要角色首次登场 | 第1-3章 |
| 世界观引入 | 介绍世界背景和规则 | 第1-3章 |
| 金手指觉醒 | 主角获得特殊能力/系统 | 第1-5章 |
| 初始冲突 | 第一个冲突事件 | 第2-5章 |
| 目标确立 | 主角确立核心目标 | 第3-5章 |
| 困境升级 | 困境进一步恶化 | 第5-10章 |

---

| 任务执行 | 执行某个任务 |
| 探索发现 | 探索新区域/发现新事物 |
